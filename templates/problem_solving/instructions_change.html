Iteration

To repeat something, but each time implement a few changes.
The method of trial-and-error with small changes each time in order to find a solution.
“iterative” doesn’t mean just repetitive. “Iterative” involves repeating something, but building on the previous iteration.
So “allow, allow” is a repetition, but “allow, arrow” (like a “word ladder” puzzle) is an iteration.
repeating a process or set of instructions again and again, each time applying it to the result of the previous stage.
