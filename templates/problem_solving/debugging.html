I can detect and correct mistakes which cause instructions (a solution) to fail (debug).

Debugging is an essential skill in problem-solving and computational thinking. It involves identifying errors in instructions, analysing their causes and making corrections to ensure that a solution works as intended.
Here is a structured approach to help detect and correct mistakes in various contexts:
1. Understand the problem
2. Before debugging clarify what the expected outcome should be.
3. Run the instructions and carefully note where and how the issue occurs.

Activities

Identify and correct errors in instructions.
  • Play "Simon Says" with a twist: Include some intentional mistakes in the instructions and see if students can spot and correct them.
  • Give students a set of jumbled instructions for a simple task (e.g., making a sandwich, building a LEGO model). Have them put the instructions in the correct order and then follow them.
    • Simple coding blocks like Scratch, intentionally insert errors for students to identify and correct
