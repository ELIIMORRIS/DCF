The ability to construct, refine, and interrogate data sets is a crucial skill in both data literacy and computational thinking. Here are some ways this competency can be developed:

  1. Constructing Data Sets
  2. Refining Data Sets

    Cleaning Data: Students can learn how to refine raw data by identifying errors, missing values, or inconsistencies. This might involve using tools like Excel’s “Find and Replace” or more advanced features like data validation.
    Data Transformation: Teachers can demonstrate techniques for transforming data into more usable formats, such as converting text to numbers, normalizing data, or sorting values.

  3. Interrogating Data Sets

    Data Analysis: Encourage students to explore data through basic statistical analysis, such as calculating averages, medians, and mode, or creating charts and graphs to visualize patterns.
    Creating Charts and Graphs: Teach students how to use software tools like spreadsheets or online chart creators to present data visually. They can create bar charts, line graphs, or pie charts to represent their findings.

Data Collection: Teachers can introduce students to methods for gathering data, whether from surveys, experiments, or external data sources like websites or open datasets. Data Sets

Data sets are organised collections of related information that can help us anseer questions or support investigations.
Data can be stored and displayed in various formats, such as:
  • Tables
  • Charts
  • Databases
  • Spreadsheets

Activities

  1. Construct a data set. Collect and organise data to answer specific questions.
  2. Refine the data set. Check the data set for accuracy in spelling and format (e.g. integers, whole numbers). Identify any missing or incorrect entries. Consider additional categories or removing unecessary categories for clear and accurate data.
  3. Interrogate the data set. Encourage curiousity and ask questions about the data set, such as "What is the highest/lowest value?" and "What patterns can we see?"
Use digital tools like Google Sheets or Excel to build data sets. Apply simple formulae to summarise data. Display the data set through bar graphs or pie charts to visually interpet the information from the data. This can help spot trends and patterns more clearly. {% endblock %}
