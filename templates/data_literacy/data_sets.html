Data Sets

Data sets are organised collections of related information that can help us anseer questions or support investigations.
Data can be stored and displayed in various formats, such as:
  • Tables
  • Charts
  • Databases
  • Spreadsheets

Activities

  1. Construct a data set. Collect and organise data to answer specific questions.
  2. Refine the data set. Check the data set for accuracy in spelling and format (e.g. integers, whole numbers). Identify any missing or incorrect entries. Consider additional categories or removing unecessary categories for clear and accurate data.
  3. Interrogate the data set. Encourage curiousity and ask questions about the data set, such as "What is the highest/lowest value?" and "What patterns can we see?"
Use digital tools like Google Sheets or Excel to build data sets. Apply simple formulae to summarise data. Display the data set through bar graphs or pie charts to visually interpet the information from the data. This can help spot trends and patterns more clearly. {% endblock %}
