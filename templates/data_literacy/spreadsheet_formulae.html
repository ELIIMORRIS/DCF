Spreadsheets are powerful tools for organising, analysing, and visualising data. They provide practical applications for real-world scenarios, like managing budgets or comparing product prices.

Spreadsheet Formulae

Basic Arithmetic Operations:
+ (Addition)
- (Subtraction)
/ (Division)
* (Multiplication)
Common Functions:
SUM: Adds up a range of cells
Example =SUM(B2:B5) will add up every value in column B, rows 2 to 5, giving the total.
AVERAGE: Calculates the mean of a range
Example =AVERAGE(C2:C5)
MAX: Finds the highest value in a range
Example =MAX(B2:B5)
MIN: Finds the lowest value in a range
Example =MIN(C2:C5)
