Formatting data is a foundational skill for data literacy.
Collecting data:
Before collecting data, define the purpose. Are there specific questions want to answer or patterns to uncover?
Is data being gathered from primary sources e.g. surveys, or secondary sources e.g. database

Tables
Tables present information or data using horizontal rows and vertical columns. Each row and column represents a specific category, labelled at the top of each column and on the left-hand side of each row. Tables are made up of cells.
Common examples of tables are:
Timetable for trains.
Class register.
Create a timetable for lessons.
Charts and Graphs
Pie Chart
Pie charts are especially useful for displaying the contrast in 2 categories e.g. yes or no, true or false. It is easier to judge which slice is bigger when there are inky 2 slices to compare.
Good for illustarting parts of a whole.
Bar Chart
Bar charts are an effective way of visualising and comparing quantities across categories.
Y-axis is vertical, X-axis is horizontal.

Spreadsheets

Formulas and functions enhance analysis capabilities.

Activities

  1. Conduct a class survey about...
  2. Enter and record this information in a table or database
  3. Organise and analyse this data into different categories
  4. Discuss why it is useful to collect and organise data and information in this way
  5. Present this information in a bar graph or pie chart. Discuss how we can better understand patterns and trends through visualising the information behind the numbers
Graphs
Graphs are graphical representations that provide a visual way to understand and analyse data, showing the relationship between two or more variables. Line Graphs are an excellent way to demonstrate change over time.

Analysis

Compare and contract data across different formats to see how data trends are shown in various representations.
Interpret the results in each format to draw informed conclusions. Look for patterns, outliers or trends that answer intitial questions.
