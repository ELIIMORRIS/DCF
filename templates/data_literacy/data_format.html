Formatting data is a foundational skill for data literacy.

Data can be:

  • Descriptive (e.g., "high" or "fast")
  • Numerical

Numerical data can be:

  • Discrete – counted (e.g., number of students)
  • Continuous – measured (e.g., temperature)

Qualitative vs. Quantitative Data

Data can be:

  • Qualitative – descriptive information (describes something)
  • Quantitative – numerical information (numbers)

Discrete vs. Continuous Data

Collecting Data

Before collecting data, define the purpose: What questions are you trying to answer, or what patterns do you want to uncover?

Data sources can be:

  • Primary (e.g., surveys)
  • Secondary (e.g., databases)

Ensure consistent units, formats, and data types are used.

Tables

Tables present information using rows and columns, with labels at the top of each column and the left-hand side of each row. Examples include:

  • Train timetables
  • Class registers
  • Lesson schedules
Charts and Graphs

Pie Chart: Useful for showing contrasts in two categories, especially with only two slices.

Bar Chart: Effective for comparing quantities across categories, with Y-axis vertical and X-axis horizontal.

Line Graph: Excellent for showing changes over time.

Tally Chart: A method for collecting information using tallies.

Databases

Databases store information in an organized way.

See more: Databases Unplugged

Spreadsheets

Spreadsheets are useful for data sorting, filtering, and basic calculations, enhanced by formulas and functions for analysis.

Activities

  1. Conduct a class survey on a topic.
  2. Record the data in a table or database.
  3. Organize and analyze the data into categories.
  4. Discuss why collecting and organizing data is useful.
  5. Present the data in a bar or pie chart to illustrate patterns and trends.
Graphs

Graphs visually represent data, allowing for analysis of relationships between two or more variables. Line graphs, for example, are ideal for showing change over time.

Analysis

Compare and contrast data across formats to observe trends, patterns, and outliers. Interpret results in each format and document your findings, observations, and insights.
