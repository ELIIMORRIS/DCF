Computational Thinking is a combination of problem-solving, scientific enquiry, and thinking skills.

Computational Thinking Skills

Can be defined by Four Key Techniques:

  • Abstraction: Simplifying complex problems by focusing on the relevant details.
    Read more

    Abstraction involves simplifying complex information by identifying and focusing on the essential and important details, and ignoring irrelevant information and unnecessary complexities. This is especially useful when facing unfamiliar problems by concentrating on what is necessary.

  • Decomposition: Breaking a problem into smaller, manageable parts.
    Read more

    Problem decomposition involves systematically dividing a complex problem into smaller, more manageable parts. This allows the assessment and resolution of each sub-problem at hand, and by deriving solutions to each sub-part, it is possible to assess all steps necessary to solve the bigger problem. For example, when working on an assisgnemnt, it is necessary to divide it into sections to answer each part individually so that it is easier to analyse and solve.

  • Pattern Recognition: Identifying similarities or patterns in data.
    Read more

    Pattern recognition involves identifying similarities and recurring themes and regularities. Recognising recurrent patterns can subsequently lead to more efficient and effective problem-solving by saving time. Pattern recognition can be seen from webiste design, where there is always a main, header, body and footer.

  • Algorithm Design: Creating a step-by-step solution to a problem.
    Read more

    An algorithm is a list of instructions needed to follow to achieve the same desired outcome every time. Algorithmic thinking involves constructung a single function that can be applied to multiple sub problems. Computers take instructions very literally, therefore instructions in the algorithm must be precise and detailed. Developing efficient algorithms is a fundamental aspect of computational thinking.

Computational thinking is a key component of the Digital Competency Framework (DCF).

Before learners can use computers to solve problems, they must first understand the problem and the methods of solving them. Through these elements, learners will understand the importance of data and information literacy, and they will explore aspects of collection, representation, and analysis.

These skills will prepare learners for the digital world and provide essential skills for the modern, dynamic workplace.
